# ⚙️ Instalação

Clone o projeto:

```bash
git clone https://github.com/seu-usuario/marie-depilacao.git
```

Entre na pasta:

```bash
cd marie-depilacao
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```
